The cotton set is intentionally limited to macro references that show:

- visible woven thread structure
- realistic jacquard integration
- natural textile lighting and depth

References that read as printed, overly smooth, or unrealistically clean should
not be added to the active production set.

`server/nanoBananaService.ts` will throw a clear runtime error if any required
reference image is missing.
